# Pokemon TCG Pocket Automation
Pokemon TCG Pocket 自動每日抽卡+領取任務

## 前置步驟
1. 安裝 BlueStacks (設定啟用 Android Debug Bridge (ADB) )
2. 安裝 Pokemon TCG Pocket on BlueStacks
3. 本機安裝 python

## 執行
```sh
cd src
python lain.py
```
